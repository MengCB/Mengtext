## 环境启动

### 安装依赖
当前目录下：
```
npm ci (推荐) 或 npm i
```

### 启动服务器
当前目录下：
```
npm run start
```

### 前端访问
当前目录启动 http-server 或直接打开 index.html  
看到浏览器控制台或后端控制台，可以看到已经连接成功。
